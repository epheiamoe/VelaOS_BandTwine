# 可视化编辑器快速入门

## 🚀 5分钟创建你的第一个故事

本指南将帮助你在5分钟内创建并导出第一个BandTwine故事。

### 第1步：启动编辑器

```bash
cd visual-editor
npm run dev
```

访问 http://localhost:5173

### 第2步：了解界面

1. **中央画布**：显示故事节点
2. **右侧面板**：编辑选中的节点属性
3. **顶部标签页**：切换不同编辑模式
4. **左侧工具栏**：添加节点等操作

### 第3步：创建第一个节点

1. 点击左侧工具栏的 **"添加节点"** 按钮
2. 新节点出现在画布中央
3. 在右侧面板编辑节点文本：
   ```
   你醒来在一个陌生的房间。
   窗外阳光明媚，鸟鸣声声。
   ```

### 第4步：添加第二个节点

1. 再次点击 **"添加节点"**
2. 将第二个节点拖到第一个节点右侧
3. 编辑文本：
   ```
   你检查房间。
   有一张床、一个书桌和一个衣柜。
   ```

### 第5步：创建连接

1. 点击第一个节点选中它
2. 在右侧面板的 **"链接"** 部分：
   - 点击 **"添加链接"**
   - 选择目标节点为第二个节点
   - 设置链接文本为 **"检查房间"**

### 第6步：添加变量

1. 切换到 **"变量管理"** 标签页
2. 在添加变量表单中填写：
   - **路径**：`player.health`
   - **初始值**：`100`
   - **描述**：`玩家生命值`
3. 点击 **"添加变量"**

### 第7步：在文本中使用变量

1. 切回 **"节点编辑"** 标签页
2. 选中第一个节点
3. 修改文本，添加变量引用：
   ```
   你醒来在一个陌生的房间。
   窗外阳光明媚，鸟鸣声声。
   
   生命值：{var.player.health}
   ```

### 第8步：添加链接动作

1. 在第一个节点的链接部分，找到刚创建的链接
2. 点击链接右侧的 **"展开动作"** 按钮
3. 点击 **"添加动作"**
4. 选择动作类型：`add`
5. 设置：
   - **目标变量**：`var.player.health`
   - **值**：`-10`
6. 点击 **"保存动作"**

现在，当玩家点击"检查房间"链接时，生命值会减少10点！

### 第9步：导出故事

1. 点击顶部导航栏的 **"文件"** 菜单
2. 选择 **"导出为 JSON"**
3. 文件将下载为 `story-data.json`

### 第10步：测试故事

1. 将导出的文件复制到 `bandtwine_src/` 目录：
   ```bash
   cp story-data.json bandtwine_src/
   ```
2. 运行编译脚本：
   ```bash
   node scripts/compile.js
   ```
3. 如果编译成功，故事就准备好了！

## 📖 下一步

现在你已经创建了基本的故事框架，可以尝试：

### 1. 添加更多节点
- 创建多个选择分支
- 设计不同的故事结局

### 2. 使用条件系统
让某些选项只在特定条件下出现：
```javascript
// 当生命值大于50时显示此选项
var.player.health.value > 50
```

### 3. 添加随机元素
让故事每次都有不同的发展：
- 添加随机选项
- 设置不同的权重

### 4. 实现完整游戏机制
- 添加显示变量（`show.`）在侧边栏显示状态
- 使用临时变量（`temp.`）进行复杂计算
- 添加图片和音效

## 💡 小技巧

1. **快速复制节点**：选中节点后按 `Ctrl+C` / `Ctrl+V`
2. **批量编辑**：按住 `Shift` 多选节点
3. **自动布局**：节点过多时，点击工具栏的"自动排列"按钮
4. **保存项目**：定期使用"导出项目"功能备份完整状态

## 🆘 遇到问题？

- **编辑器打不开**：检查端口是否被占用，尝试 `npm run dev -- --port 5174`
- **节点不显示**：刷新页面，或检查浏览器控制台错误
- **导出失败**：确保所有链接都有有效的目标节点
